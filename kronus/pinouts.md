## Bootstrap pins

| SPI_CSZ | SPI_DI |             mode             |
|---------|--------|------------------------------|
|    1    |    0   | Normal boot (EJTAG disabled) |
|    1    |    1   | Normal boot (EJTAG enabled)  |
|    0    |    0   | SBUS1                        |
|    0    |    1   | DBUS1                        |
